# HTML Slide Editor Engine

Shared CSS and JavaScript engine for deck-specific HTML slide files.

## Usage

Use these URLs from deck HTML files after enabling GitHub Pages:

```html
<link rel="stylesheet" href="https://zoselotta-maker.github.io/html-slide-editor/v1.0/slide-editor.css">
<script src="https://zoselotta-maker.github.io/html-slide-editor/v1.0/slide-editor.js"></script>
```

Deck-specific content should stay inside the `deck-data` JSON block in each HTML file.
